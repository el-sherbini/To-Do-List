# To-Do App
> Simple to-do app.
> Live demo [_here_](https://bre.is/4LMHNtkb).
## Technologies Used
- Pug.js
- Sass
- JavaScript
## Features
- Add to-do item with creation date
- Delete to-do item
- Mark completed to-do item
- Get count of total and completed to-do items
- Save your data in browser local storage
